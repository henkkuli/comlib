    Self: Sized, {
    type InputError;

}
Expand description

Value which can be greedily consumed from &str.

Associated Types

[src]

type InputError

Type for representing errors which can happen during consuming values.

Required methods

[src]

fn consume(input: &str) -> Result<(Self, &str), Self::InputError>

Consume input from the given &str.

In case of success, returns the consumed value and the rest of the string. In case of an error, returns the error.
